# Personal Portfolio — Shivam Sharma

## Overview
A modern, responsive personal portfolio built with semantic HTML5 and CSS3. It showcases projects, skills, contact details, and a downloadable resume. This project uses no JavaScript or CSS frameworks — just plain HTML and CSS with responsive techniques.

## Features
- Clean, semantic HTML structure using `<header>`, `<main>`, `<section>`, and `<footer>`.
- Responsive layout built with CSS Grid and Flexbox.
- Animated skill progress bars using pure CSS.
- Project cards, resume download, and contact form (front-end only).
- Google Font: Poppins.
- Modern UI with blue palette, soft shadows, rounded cards, and hover effects.

## Folder Structure
- `index.html` — Main page
- `style.css` — All styles
- `images/` — Placeholder images (SVGs) for profile & projects
- `resume.pdf` — Placeholder resume file
- `README.md` — This document

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, animations)
- Google Fonts

## How to view the project
1. Clone or download the project files.
2. Open `index.html` in a web browser (double-click or use "Open with" → Browser).

No build tools or local servers are required — this is a static site.

## Customization
- Replace images in `images/` with your own photos and project screenshots.
- Update contact details in `index.html`.
- Replace `resume.pdf` with your actual resume (keep the filename the same to preserve links).
