# Simple Landing Page Project

This is a beginner-friendly landing page assignment created using only HTML and CSS.

## Files
- index.html - main page structure
- style.css - styling for the page
- images/placeholder.svg - simple placeholder image

## How to Open
Open index.html in your browser to view the website.

## Features
- Simple header and navigation
- Hero section with button
- About section
- Services cards
- Contact form
- Responsive design for mobile and desktop
